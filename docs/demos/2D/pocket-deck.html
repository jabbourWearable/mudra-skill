2
The Problem

A clicker is a tether.

Slide-decks pull the speaker back to a device. The hands stop conducting and start fumbling. The room feels it within five seconds.

3
The Gesture Language

Two motions.
Whole deck.

  • Pinch (thumb-to-index tap) — advance to the next slide.
  • Double-tap — go back one slide. Same finger, faster cadence.
  • That's it. No menu, no mode, no second hand.
4
Always-On Affordance

A thin line
tracks the talk.

A coral-to-gold progress bar sits across the top of the stage. The speaker glances up; the audience sees pace, not panic. The bar fills slide-by-slide — never scrubbed, never noisy.

5
Fallback By Design

If the band sleeps,
the keyboard stays.

  • Space or → — next slide.
  • B or ← — previous slide.
  • Same handler path. The deck never knows the difference.
